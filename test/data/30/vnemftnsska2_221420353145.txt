저는 다운로드하기 전에 database라고 폴더 새로 생성한 뒤에 넣어줬습니다.다운로드한 파일 2개 압축을 풀었으면 2of2에 있는 파일 4개를 1of2로 복사해서 넣어줘야 합니다.
database &gt; stage &gt; Components 경로 적은 거 보이시죠? 보이는 것처럼 실행해주시면 됩니다.이제 setup 파일 실행시켜주면 오라클(Oracle) 설치가 시작됩니다!중간에 이런 메시지가 떠서 안되는 거 아닌가 했는데 좀 기다리니까 실행됐습니다. 다시 읽어보니 시작 준비 중이라네요. 그냥 넘기셔도 돼요!처음 화면인데 이메일 적는 부분 무시하셔도 됩니다. 그냥 아래 다음 버튼 누리시면 우측의 화면처럼 나오는데 이 부분도 예 버튼 누르시면 돼요.이쪽 화면도 뭐 특별한 거 없이 그냥 넘어가면 됩니다. 좌측 화면에서 다음 버튼 누르면 우측 화면 나오는데 다음 누르시면 됩니다.이 화면에서는 경로 바꾸실 분들은 바꾸셔도 되지만 저는 그냥 내버려 두는 거 추천해요. 나중에 경로 못 찾고 헷갈리는 상황이 있을 수도 있고 비밀번호의 경우엔 학원에서 admin으로 다 통일해서 했던 걸로 기억하는데 자주 쓰는 비밀번호로 적어주시면 됩니다. 어차피 연습용으로 사용할 계획이라 잘 기억에 남는 거 적으시고 권장 표준을 따르지 않는 비밀번호라고 나와도 그냥 무시하고 '예' 버튼 누르시면 됩니다.번호 빨간색 글씨로 적어뒀습니다. 필요조건 검사하시고 완료 누르시면 되는데 저 같은 경우엔 필요조건 검사가 클릭이 안 되길래 그냥 넘어갔습니다.
아무 문제 없이 오라클 잘 작동합니다.이 화면 순서대로 진행하시면 되고요... 데이터베이스 생성이 완성되면 '비밀번호 관리 버튼'을 클릭하셔야 해요!이거 클릭하셔서 scott 부분 잠금 해제하셔야 연습용 DB를 활용하실 수 있습니다.비밀번호 설정에 대한 경고문 뜰 때마다 예 버튼 눌러서 넘겨줬는데 혹시 모르니 자세히 읽어보시고 넘겨주세요 ~이로써 오라클 설치가 끝났습니다.
이제 설치가 제대로 작성이 되었는지  cmd 창을 통해 확인해보도록 합니다. Test는 필수!

밑에 검색 버튼 누르신 다음 cmd라고 적으시면 명령 프롬프트 cmd 실행 아이콘이 나옵니다.

cmd 창이 떴으면 데이터베이스에 접속을 해야 하는데 이렇게 적어주시면 됩니다.

sqlplus "/as sysdba"

띄어쓰기까지 잘 해주셔야 해요. sqlplus 띄고 "/as 띄고 sysdba"라고 적으시고 엔터를 누르시면 Oracle Database에 접속되었다고 화면에 뜹니다.
그리고 경로가 SQL&gt; 로 바뀐 거 보이시죠? 그럼 여기서 scott 계정으로 접속하셔서 scott 계정에 있는 DB들을 사용하시면 됩니다.
SQL&gt; conn scott/tiger
보이시죠? scott 계정으로 연결하라는 뜻입니다. tiger는 scott 계정의 비밀번호입니다. Database 설치 중에 계정 비밀번호 설정하는 부분 위에 있습니다.
conn scott만 치면 비밀번호 입력하라는 메시지가 뜨는데 그때 비밀번호를 적어도 상관없습니다.

conn 명령을 통해 scott 계정에 연결이 되었다면 sql 문을 적어 scott 계정의 DB가 잘 나오는지 확인하면 됩니다.
저는 scott 계정의 테이블 명을 학원에서 실습하면서 바로 select * from emp; 라고 적었지만 scott 계정이 가지고 있는 테이블을 보고 싶다면?
select * from tab; 이라고 적으시면 scott이 가진 테이블 목록을 보실 수 있습니다. 오라클 설치 완벽하게 된 거 확인했습니다 :)
